# 1.4.1 — Easy Way to Know if the Baby's Meal is "Set" (and the Weight Connection)

## When is a meal "set"?
A meal is **set** when the child eats **3–4 bites with visible excitement/happiness**. This applies to **breakfast, lunch and dinner**. The rule "if lunch is set, start breakfast; if breakfast is set, start dinner" depends on this.
- **Snacks do not need to be set** — they are optional and may be skipped (e.g., if the child is asleep).
- **Breakfast, lunch and dinner must NOT be skipped.**

## "Set" also means fixed timing windows
- **Breakfast:** 8–9:30 am
- **Lunch:** 12–2 pm
- **Dinner:** 7–8:30 pm
- A ±30-minute shift by the child's hunger is fine. Don't let a missed nap push lunch to 3 pm — never repeat that.

## Customized meal setting
If a child eats breakfast and lunch well but not dinner: offer dinner earlier (~6:30 pm). If the child eats dinner very early, top up around 7:30 pm with simple protein — **paneer cubes, cheese cube, chicken piece, egg white**. Set meals around the child's own routine.

## Meal-set vs. weight — expected weekly weight gain
- **0–3 months:** ~**150 g/week** if exclusively breastfed (more if combination/formula fed).
- **3–6 months:** ~**130 g/week**.
- **At 5 months complete:** birth weight should have **doubled** (use **corrected age** for premature babies — e.g., born 2 weeks early → double 2 weeks later; corrected age is used **everywhere** in the course for premature babies, including food-introduction ages such as egg at ~8.5 months instead of 8).
- **6–9 months:** ~**110 g/week**.
- **9–12 months:** ~**90–100 g/week**.
- **First birthday:** weight should be roughly **triple** the birth weight.

## Normal weight "stagnation"
Somewhere in the 6–12 month window there will be a **1–2 month phase where weight doesn't change** — this is **completely normal**. It happens when the child's energy shifts to a new skill (teething, rolling over, sitting, neck control).
- **Weight stagnant = allowed.**
- **Gradual weight gain = the goal.**
- **Weight loss = not allowed**, except with severe loose motions / vomiting / illness (then ~500 g loss can happen).

## After the first birthday
Plans change (the toddler runs off, gets distracted) — that's a **separate toddler course** (register on the website). For now, this meal-setting/planning/weight framework holds **for the first year**.
